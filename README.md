# 🌍 Pack Your Bags

A full-stack travel booking web application built using HTML, CSS, JavaScript, and Firebase.

---

## 🚀 Features

- 🔐 User Authentication (Firebase Auth)
- 👤 Username-based Login
- 🗂 Firestore Database Integration
- 🛒 Cart System with Quantity Logic
- 👤 User Profile System
- 🔒 Protected Routes
- 📦 Dynamic Cart Data per User

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Firebase Authentication

- Git & GitHub

## 🔥 How It Works

1. Users sign up using email & password.
2. Username is stored separately in Firestore.
3. Cart items are saved inside:
   users → userUID → cart (subcollection)
4. Quantity updates dynamically.
5. Cart total calculates automatically.
